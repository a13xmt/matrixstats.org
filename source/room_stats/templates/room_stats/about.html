About

The project goals are to provide the best user experience for exploring the Matrix ecosystem. We belive that Matrix protocol is the best option for instant messanging and it would become a standard in the future. However, by the end of summer 2017 there wasn't a lot of public catalogs presented, so matrixstats.org was started. We aim to make rooms more searchable and descriptive, so the users can explore thousands of rooms and become a part of the network shortly. Please, feel free to describe your experience or propose some features you may find interesting.

Some quick notes:

  • The huge amount of rooms is not categorized yet, interface for category changes/proposals will come soon
  • Mobile interface is broken right now, should be fixed in the later releases
  • Room list items are too big right now. I'm searching a way to greatly reduce them.
  • Community room promotions would be free ofc.
  • Ratings page should be redesigned later.
  • Room members graph is showing only last month statistics right now. It should be improved.
